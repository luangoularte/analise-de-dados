# Análise de Dados
Tecnologias
- Python
- Jupyter

Neste projeto, o objetivo foi identificar o perfil do cliente ideal para os produtos de uma empresa fictícia, para executar essa tarefa foi recebido uma tabela com milhares de clientes, cujo o processamento e análise foi usado o programa para esse fim. Primeiramente foi realizado o tratamento de dados da tabela em questão para evitar complicações e erros. Posteriormente foi gerado os gráficos, comparando a coluna nota com as demais colunas da lista, fornecendo informações para encontrar o cliente modelo. Por fim, todos os dados são coletados e é elaborado as considerações finais sobre o consumidor alvo.

Após a análise minuciosa das tabelas podemos ter as seguintes considerações:

Perfil ideal de cliente
- Acima de 15 anos (não tem muita diferença entra as faixas etárias depois disso)
- Faixa salarial não parece fazer tanta diferença
-  Áreas de trabalho: Entretenimento e Artista (evitar Construção)
-  Tem entre 10 e 15 anos de experiência
-  Com familías não tão grandes (até no máximo 7 pessoas)
